Politique de confidentialité

Dernière mise à jour : 20 mars 2026

1. Données collectées

Offrii collecte les données suivantes lors de votre utilisation de l'application :

  • Compte : adresse email, nom d'affichage, photo de profil (optionnelle)
  • Listes de souhaits : vos envies, catégories, priorités, liens et images associés
  • Cercles : les groupes que vous créez ou rejoignez, et les membres associés
  • Entraide : les besoins communautaires que vous publiez ou auxquels vous contribuez
  • Messages : les messages échangés dans le cadre de l'entraide

2. Utilisation des données

Vos données sont utilisées exclusivement pour :

  • Le fonctionnement de l'application (authentification, partage, notifications)
  • L'envoi d'emails transactionnels (vérification, réinitialisation de mot de passe)
  • La modération du contenu communautaire (via analyse automatisée)

3. Partage des données

Vos données ne sont jamais vendues. Elles sont partagées uniquement avec :

  • Les membres de vos cercles : vos envies partagées et votre profil public
  • Les utilisateurs de l'entraide : vos besoins publiés et messages associés
  • Nos sous-traitants techniques : hébergement (Hetzner, Allemagne), stockage (Cloudflare R2), emails (Resend)

4. Stockage et sécurité

Vos données sont stockées sur des serveurs sécurisés en Europe (Hetzner, Allemagne). Les communications sont chiffrées en transit (TLS). Les mots de passe sont hachés avec Argon2.

5. Vos droits

Conformément au RGPD, vous pouvez :

  • Accéder à vos données depuis votre profil dans l'application
  • Modifier ou supprimer vos données à tout moment
  • Supprimer votre compte (suppression définitive de toutes vos données)
  • Nous contacter : contact@offrii.com

6. Connexion via Google / Apple

Si vous utilisez la connexion Google ou Apple, nous recevons uniquement votre adresse email, votre nom et votre photo de profil. Nous n'accédons à aucune autre donnée de votre compte Google ou Apple.
